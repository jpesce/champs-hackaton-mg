Sobre

O Dataminas é resultado de um Hackathon organizado pelo Movimento Minas como iniciativa para facilitar o acesso a todos às informações do Portal Transparência. Nossa proposta é permitir a visualização temporal dos dados e chamar a população para participar e usar esses dados.

O site foi desenvolvido em Django, os gráficos foram gerados com a biblioteca D3. O código para gerar esse site está disponível no GitHub.

Nosso modelo para detectar anomalias é bem simples (mas um pouco técnico):

Primeiro, nós encaixamos um modelo polinômial nos dados para obter um valor que seria o esperado de cada mês. Nós comparamos o valor esperado ao valor que foi realmente efetuado, e assinalamos ao ponto um escore de 1 a 100, sendo 1 bem próximo do valor esperado, e 100 bem distante. Para ser mais preciso, um escore de 100 ocorrerá 0.1% das vezes caso o erro do modelo seja normalmente distribuído. Nós sinalizamos sempre que esse escore ultrapassa 60.
